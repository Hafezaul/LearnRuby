3. `array.first()` & `array.last()`

## A. `array.first()` 

Digunakan untuk mengambil value awal dalam sebuah array. 

Contoh 
```
arr1 = [1, 2, 3, 4, 5, 6, 7]

p arr1.first 
# Output yang diambil ialah ' 1 '
```

## B. `array.last()`

Digunakan untuk mengambil value akhir dalam sebuah array. 

Contoh 

```
arr1 = [1, 2, 3, 4, 5, 6, 7]

p arr1.last
# Output yang diambil ialah ' 7 '
```
